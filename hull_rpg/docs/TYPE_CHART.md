# TYPE CHART (v0.1)

Types used: Water, Grass, Electric, Earth, Steel, Psychic, Air, Light.

Key advantages:
- Water > Fire/Earth
- Grass > Water/Earth
- Electric > Water/Air
- Earth > Electric/Steel
- Air > Grass
- Light > Psychic

Implementation lives in `scripts/managers/BattleManager.gd` as a compact multiplier map.
